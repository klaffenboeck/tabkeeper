# Tabkeeper

A gem to help you keep track of your active state for tabs. 

Please note: This gem is not yet production ready, it is in a very early development stage. 

## Installation

Add this line to your application's Gemfile:

    gem 'tabkeeper'

And then execute:

    $ bundle
